# 实时视频换脸 + 声音模仿 接入指南

> 目标：将换脸+变声效果实时接入微信视频/抖音直播/Zoom等任意视频APP

---

## 整体架构

```
【视频链路】
  物理摄像头 / 手机(scrcpy)
        ↓
  realtime_stream.py（换脸处理）
        ↓
  OBS Virtual Camera（虚拟摄像头）
        ↓
  微信/抖音/Zoom → 选择"OBS Virtual Camera"

【音频链路】
  物理麦克风
        ↓
  RVC 实时变声（go-realtime-gui.bat）
        ↓
  VB-Cable 虚拟音频线
        ↓
  微信/抖音/Zoom → 选择"CABLE Input (VB-Audio Virtual Cable)"
```

---

## 一、安装前置软件（一次性）

### 1. OBS Studio（虚拟摄像头驱动）
- 下载地址：https://obsproject.com/
- 安装完成后，打开 OBS → 顶部菜单「工具」→「虚拟摄像头」→「启动」
- 之后每次开机执行一次即可（或设置开机自启）
- ✅ 安装后，系统出现「OBS Virtual Camera」摄像头设备

### 2. VB-Cable（虚拟音频线）
- 下载地址：https://vb-audio.com/Cable/
- 解压后右键以管理员运行 `VBCABLE_Setup_x64.exe` → Install Driver
- ✅ 安装后，系统出现「CABLE Input」麦克风 和「CABLE Output」扬声器

## 二、视频换脸接入步骤

### 第一步：在 AvatarHub 选好角色
打开 http://127.0.0.1:9000/ui
1. 新建角色（上传你想变成的人脸图片）
2. 点击「激活」该角色

### 第二步：启动实时换脸
双击 `start_realtime.bat`，选择视频输入源：
- 选 `1`：使用本机 USB 摄像头
- 选 `3`：使用手机摄像头（需先开 scrcpy）

### 第三步：在视频APP中选择虚拟摄像头
| 应用 | 设置位置 |
|------|---------|
| **微信** | 视频通话中 → 右下角设置 → 摄像头 → 选「OBS Virtual Camera」|
| **抖音直播** | 开播设置 → 视频源 → 选「OBS Virtual Camera」|
| **Zoom** | 设置 → 视频 → 摄像头 → 选「OBS Virtual Camera」|
| **腾讯会议** | 设置 → 视频 → 摄像头 → 选「OBS Virtual Camera」|
| **OBS直播** | 来源 → 添加「视频捕获设备」→ 选「OBS Virtual Camera」|

## 三、声音变声接入步骤

### 第一步：安装 RVC 所需依赖
```bat
cd C:\模仿音色\Retrieval-based-Voice-Conversion-WebUI
go-realtime-gui.bat
```

### 第二步：RVC 实时变声界面操作
1. 打开 RVC 实时变声界面后：
   - `Input Device`：选你的真实麦克风
   - `Output Device`：选「CABLE Input (VB-Audio Virtual Cable)」
   - `Model`：选你训练好的声音模型
   - 点击「Start Voice Changing」

### 第三步：在视频APP中选择虚拟麦克风
| 应用 | 设置位置 |
|------|---------|
| **微信** | 视频通话 → 设置 → 麦克风 → 选「CABLE Output (VB-Audio Virtual Cable)」|
| **抖音直播** | 开播设置 → 音频输入 → 选「CABLE Output」|
| **Zoom** | 设置 → 音频 → 麦克风 → 选「CABLE Output」|

## 四、命令行参数说明

```bat
# 使用0号摄像头，720p
python realtime_stream.py --source 0 --width 1280 --height 720

# 使用scrcpy手机画面
python realtime_stream.py --source scrcpy

# 使用RTSP网络摄像头
python realtime_stream.py --source rtsp://192.168.0.100:8554/stream

# 无预览模式（直播时节省性能）
python realtime_stream.py --source 0 --no-preview
```

## 五、性能参考（RTX 4070）

| 模式 | 换脸延迟 | 虚拟摄像头输出 |
|------|---------|--------------|
| 720p@5fps换脸 | ~200ms | 30fps流畅 |
| 1080p@3fps换脸 | ~350ms | 30fps流畅 |

> 视频输出始终 30fps 流畅，换脸帧率决定「画面更新频率」  
> 换脸间隔内自动使用上一帧换脸结果，无卡顿感

## 六、常见问题

**Q: OBS Virtual Camera 不出现在APP里**  
A: 重启 OBS，在工具→虚拟摄像头里点「启动」，再重启目标APP

**Q: 变声有延迟**  
A: RVC 界面中将 `Response Threshold (ms)` 调小，或降低 Pitch Shift

**Q: 换脸不生效（显示原始画面）**  
A: 检查 AvatarHub 是否有激活角色；检查 FaceSwap API 是否运行

**Q: 抖音/微信看不到虚拟摄像头**  
A: 以管理员权限运行 OBS，再启动虚拟摄像头
