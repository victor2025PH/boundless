安装虚拟声卡（VB-Cable）

虚拟声卡是把变声后的声音送进直播软件(OBS)的必经通道——没有它观众听不到声音。装一次即可，全程约 2 分钟。

  1. 1 下载免费安装包：vb-audio.com/Cable（VBCABLE_Driver_Pack）
  2. 2 解压后右键 VBCABLE_Setup_x64.exe →「以管理员身份运行」→ 点 Install Driver
  3. 3 重启电脑（声卡驱动需重启后生效）
  4. 4 打开 Windows 声音设置，确认 CABLE Input 未被禁用
  5. 5 回到本页，点下面「已装好，重新检测」
💡 已经装过？多半是设备被禁用：Windows「设置 → 系统 → 声音」里确认 CABLE Input 未被禁用，再点重新检测。
