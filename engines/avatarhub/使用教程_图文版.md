# 使用教程（图文版）

> 面向**日常使用**：装好之后，按“我想做什么”找到对应场景，照着点。
> 已装好？没装先看《安装教程（图文版）》。运维/排错见 `使用说明.md`。

---

## 打开哪个页面

| 我想… | 打开 |
|---|---|
| 管理角色/声音、开播、看状态（电脑主控台） | `http://127.0.0.1:9000/ui` |
| 跟数字人对话（电脑） | `http://127.0.0.1:9000/converse` |
| 跟数字人对话 / 手机同传（手机） | `http://<本机IP>:9000/phone` |
| 手机当麦克风+摄像头做同传/出镜 | 见下方场景③ |

> `<本机IP>` 在主控台顶部或 `/setup` 页会显示，例如 `192.168.1.49`。手机要和电脑连**同一个 WiFi**。

---

## 场景① 克隆一个声音 + 跟数字人对话

1. 主控台 `/ui` → **角色/声音** → 上传一段 **10 秒左右清晰人声**（少杂音、别太短）。
2. 点 **克隆** → **试听**，满意就 **激活**这个角色。
3. 进 `/converse`（电脑）或 `/phone`（手机）→ 打字或按住麦克风说话。
4. 数字人用**克隆音 + 口型**实时回应，支持中途打断（barge-in）。

> 手机端第一次按麦克风会弹“允许使用麦克风”，点 **允许**。误点“禁止”→ 地址栏🔒→网站设置→麦克风→允许→刷新。

---

## 场景② 直播/视频换脸出镜（微信/抖音/Zoom）

1. 装好 **OBS Studio**（虚拟摄像头）和 **VB-Cable**（虚拟声卡）——见 `实时直播接入指南.md`。
2. 主控台 `/ui` → 新建/激活一个**出镜角色**（上传要变成的人脸）。
3. 开播页 → **🎛 设备体检**（麦/摄像头/虚拟声卡三盏灯）→ 绿了点 **一键开播**。
4. 在微信/抖音/Zoom 里，把**摄像头**选成「OBS Virtual Camera」、**麦克风**选成「CABLE Output」。

> 没有物理摄像头？用手机当摄像头——见场景③或 `手机音视频输入指南.md`。
> 画面比例：手机竖屏视频通话选「竖屏 720×1280」。

---

## 场景③ 手机同传（说中文→对方听到你的克隆音外语）

**这是“扫码授权”最容易卡住的一条，单独出了详细图文：见《手机同传扫码授权全流程》。** 极简版：

1. 电脑：`boot_stack.bat`（或启动器“启动全部”），确认「实时同传 (通译 LingoX)」运行中。
2. 电脑：浏览器开 `http://<本机IP>:7878/show`，看到两个码——**做同传扫【① 做同传】**。
3. 手机：连同一 WiFi 扫①→ 提示“连接不是私密”→ **高级/显示详情 → 继续前往/访问此网站**（本机自签证书，安全）。
4. 手机：点 **🚀 一键准备** → 麦克风 **允许** → 顶部引导条变绿 = 可以说话。
5. （出镜）点 **📷 开摄像头** → 允许 → 电脑换脸源填 `http://127.0.0.1:7878/cam.mjpeg`。

> 手机页顶部有一条会自我判断的“下一步”引导条：该干嘛它会告诉你；点错“禁止”它会给分浏览器的恢复步骤。

## 场景④ 会议纪要 / 给录像配字幕

主控台 `/ops` → **同传转写 / 字幕导出**：选会话与说话人，一键导出 TXT / SRT / JSON。往期会话结束后自动留存。

## 常用小抄

| 想做 | 怎么做 |
|---|---|
| 只重启控制中心（改了页面） | 见 `使用说明.md` §3.3（秒级，不动显存） |
| 一键体检 | `doctor.py` 或启动器「一键体检」 |
| 手机收告警 | `/ui` 配企业微信/钉钉 webhook（`使用说明.md` §6.4） |
| 换 WiFi 后手机连不上 | 扫码页会自动换新码，手机重扫即可 |
| 延迟大 | 用 5GHz WiFi；同传面板可切“低延迟模式” |

## 出问题先看这里

| 现象 | 处理 |
|---|---|
| 页面打不开 | 看 hub 是否运行；硬刷新 Ctrl+F5 |
| 手机连不上 | 手机/电脑同一 WiFi；管理员跑 `_allow_firewall_7878.bat` |
| “连接不是私密” | 正常，自签证书，继续前往/访问此网站 |
| 麦克风开不了 | 地址栏🔒→网站设置→麦克风→允许→刷新 |
| 同传没字幕 | 电脑端通译 LingoX(7900) 要开着 |
| 换脸显示原图 | 确认有激活角色 + 换脸服务在跑 |

更细的排错见 `使用说明.md` §8 和《手机同传扫码授权全流程》§8。
