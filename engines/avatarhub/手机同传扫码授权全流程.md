# 手机同传 · 扫码授权全流程（图文版）

> 面向**终端用户/主播/业务**：一台手机当麦克风+摄像头，说中文→对方听到你的**克隆音**外语，屏幕看中英字幕。
> 本文只讲“**扫码之后每一步点哪里**”，把「选哪个码 / 证书告警 / 麦克风摄像头授权」三个卡点一次讲透。
> 配套：整机安装见 `部署指南.md`；日常运维见 `使用说明.md`；DroidCam/OBS 另一条链路见 `手机音视频输入指南.md`。

---

## 0. 一分钟看懂：这条链路在干嘛

```
你（手机麦克风）──说中文──▶ 电脑：识别→翻译→你的克隆音配音 ──▶ 对方（微信/Zoom…）听到外语
                                     │
对方说话 ──▶ 电脑同传翻译 ──字幕──▶ 你的手机屏幕（中英双列，边说边看）
（可选）手机前置摄像头 ──▶ 电脑换脸 ──▶ 虚拟摄像头出镜
```

- **手机端**只做三件事：**当麦克风**、**放对方声+看字幕**、（可选）**当摄像头**。
- 重活都在电脑。手机只要连**同一个 WiFi** + 扫一次码 + 授权一次。

---

## 1. 电脑端：先把服务打开（一次性）

同传要两个服务活着：**通译 LingoX（翻译，端口 7900）** 和 **手机中继（端口 7878/7879）**。

**推荐**：双击 `launcher.bat` → 勾「启动全部」，或命令行：

```bat
:: A. 一次拉起同传全家桶（含 hub / 翻译 / 中继）
boot_stack.bat

:: B. 或分别拉：先翻译，再手机中继
_launch_interp.bat
start_monitor_relay.bat
```

> - `_launch_interp.bat` = 通译 LingoX（7900，字幕/翻译的真正来源）。
> - `start_monitor_relay.bat` = 手机中继（7878 http 只听 / 7879 https 对讲）。首次启动会**自动生成自签证书**（手机采集麦克风/摄像头必须走 https）。
> - 首次用手机连，如果连不上，**以管理员**运行一次 `_allow_firewall_7878.bat` 放行 7878/7879。

**确认服务在线**：浏览器开 `http://127.0.0.1:9000/ui`，右侧「📊 服务状态」里「实时同传 (通译 LingoX)」为运行中即可。

---

## 2. 让电脑显示二维码

任选其一：

- **方式一（推荐）**：`http://127.0.0.1:9000/ui` → 开播页 →「📱 手机无线开播」向导，里面有二维码 + 一键无线开播。
- **方式二（最直接）**：电脑浏览器打开 **`http://<本机IP>:7878/show`**（本机 IP 在页面顶部会显示，例如 `192.168.1.49`）。

`/show` 页会看到**两个码，按“你要做什么”选一个**：

| 码 | 什么时候扫 | 说明 |
|---|---|---|
| **① 做同传 · 推荐**（🎤 手机当麦克风+摄像头） | 你要**开口说话**、让对方听到你的克隆音、或出镜换脸 | 走 https（安全页），能采集麦克风/摄像头 |
| ② 只旁听（🎧 只听对方+看字幕） | 你**不说话**，只想听对方声、看字幕 | 走 http，免证书最省事 |

> 做同传请扫 **①**。换 WiFi / IP 变了，页面会自动刷新二维码并提示“请重新扫码”，重扫①即可。

---

## 3. 扫①之后：手机会提示“连接不是私密” —— 这一步别退出

因为采集麦克风/摄像头必须走 https，而这是**本机自签证书**（不是真被劫持），手机浏览器会拦一下。**这是正常的，按下面点“继续”：**

| 你的手机/浏览器 | 怎么点 |
|---|---|
| **安卓 · Chrome** | 点「**高级**」→「**继续前往（不安全）**」 |
| **iPhone · Safari** | 点「**显示详细信息**」→「**访问此网站**」→ 再确认一次 |
| **微信里打开的** | 点右上「**···**」→「**在浏览器打开**」（**微信不给麦克风/摄像头权限**，必须转到系统浏览器） |

> 记忆点：**“高级/显示详情” → “继续前往/访问此网站”**。一次通过后，同一台手机同一 WiFi 下次一般不再拦。

### 进阶：一劳永逸免告警（可选，装一次证书）

嫌每台手机首次都要点“继续前往”麻烦？可以给手机装一次本机证书，之后对讲页**永不再弹告警**（换 WiFi/IP 也不用重装）：

1. 电脑 `/show` 扫码页上「B · 一劳永逸免告警」区有个小二维码，手机扫它（走 http，不会有告警）→ 打开**证书安装页** `http://<本机IP>:7878/cert`。
2. 按手机类型：
   - **iPhone**：下载描述文件 → 设置顶部「已下载描述文件」→ 安装 → **关键**：设置 → 通用 → 关于本机 → 证书信任设置 → 打开「AvatarHub Local CA」。
   - **安卓**：下载 `avatarhub-ca.crt` → 设置 → 安全 → 加密与凭据 → 安装证书 → CA 证书 → 选它。
3. 装完回对讲页，`https://<本机IP>:7879` 直接打开、无告警。

> 原理：本机生成一个只在你电脑上的**根证书（CA）**，同传页证书由它签发；手机信任这个 CA 一次即可。私钥不外发，仅用于本机局域网。不想装也没关系——§3 的「继续前往」永远可用。

## 4. 进入手机页后：跟着顶部“下一步”引导条做

页面顶部有一条会**自动判断你到哪一步**的引导条（蓝=待办 / 绿=就绪 / 红=要处理）。照它做即可：

### 第 1 步 · 点【🚀 一键准备】
- 引导条会高亮闪烁下方绿色的 **🚀 一键准备**。点它。
- 它自动完成：选对方声输出设备 → 开监听音频+字幕 → 启动同传会话 → 请求麦克风。

### 第 2 步 · 允许麦克风
- 手机弹「**是否允许使用麦克风？**」→ 点 **允许**。
- 允许后引导条变绿：“**就绪，直接对着手机说话**”。这时你说话，屏幕就出中英字幕、电脑用你的克隆音把外语播给对方。

### 第 3 步（可选）· 要露脸出镜再允许摄像头
- 只做**语音同传**：跳过，不用开摄像头。
- 要**换脸出镜**：点下方 **📷 开摄像头** → 弹窗点 **允许**。电脑换脸源填 `http://127.0.0.1:7878/cam.mjpeg`。

> **顶部五盏灯全亮 = 全通**：音频 / 字幕 / 同传 / 麦 / 摄像头。灯的含义见 §7。

## 5. 开始使用：按钮速查

| 按钮 | 作用 | 什么时候用 |
|---|---|---|
| 🚀 一键准备 | 按推荐顺序自动把上面几步做完 | **第一步永远先点它** |
| ▶ 开始监听 | 把电脑里“对方的声音”放到手机 | 想用手机边听边看字幕 |
| ▶ 开始同传 | 启停电脑上的翻译会话（字幕来源） | 一键准备已含；单独启停时用 |

| 🖐 常开 | 常开麦，检测到说话自动发送 | 长时间对话（推荐） |

| 📷 开摄像头 | 把手机摄像头推给电脑换脸 | 要出镜时 |

## 6. 权限点错了怎么办（误点“禁止”的恢复）

一旦误点“禁止”，浏览器会**记住**，之后点一键准备也开不了麦。引导条会变**红**并直接给出对应恢复步骤：

| 手机/浏览器 | 恢复麦克风 / 摄像头 |
|---|---|
| **安卓 · Chrome** | 点地址栏左侧 **🔒 / ⓘ 图标 → 网站设置/权限** → 把麦克风/摄像头改「允许」→ **刷新** |
| **iPhone · Safari** | **设置 → Safari → 相机/麦克风** 改「允许」，或地址栏 **ᴀA → 网站设置** 打开 → **刷新** |
| **微信里** | 微信不给权限：右上 **···→ 在浏览器打开**，重走 §3–§4 |

> 恢复后点引导条里的「**我已允许，刷新**」即可。

## 7. 顶部状态灯含义

| 灯 | 绿=正常 | 不亮/黄/红时 |
|---|---|---|
| 音频 | 已在放对方声 | 点「▶ 开始监听」；或电脑没在出声 |
| 字幕 | 字幕流已连 | 见「同传」灯；网络抖动会自动重连 |
| 同传 | 通译 LingoX 运行中 | **离线** = 电脑端没开 7900：去电脑跑 `_launch_interp.bat` |
| 麦 | 手机麦在发送 | 待机=没开；被禁=看 §6 |
| 摄像头 | 已推流到电脑 | 待机=没开摄像头（语音同传不需要） |

## 8. 常见问题（按现象查）

**Q1 扫码后手机一直转圈/打不开**
先确认手机和电脑**连同一个 WiFi**（不是一个连 WiFi 一个连热点）；仍不行→电脑上**以管理员**运行 `_allow_firewall_7878.bat`；公司网络隔离了设备互访时换成手机热点让电脑连。

**Q2 “连接不是私密/证书无效”不敢点**
正常，见 §3，这是本机自签证书。点“高级→继续前往 / 显示详情→访问此网站”。

**Q3 页面进去了，但没字幕**
顶部「同传」灯是不是离线？离线=电脑端翻译服务没开：电脑跑 `_launch_interp.bat`（或 `boot_stack.bat`），等「服务状态」里“实时同传”变运行中，手机刷新。

**Q4 我说话，对方没听到 / 没克隆音**
①「麦」灯是否在发送；②电脑端把通话软件的麦克风设成虚拟声卡（一键通话会自动做）；③克隆音色没配时会“仅字幕”，电脑端角色要绑好克隆声音。

**Q5 延迟有点大**
优先 5GHz WiFi、手机靠近路由器；电脑端在同传面板可切“低延迟模式”（牺牲一点音色相似度换速度）。

**Q6 换了 WiFi 之后连不上**
`/show` 页会自动换新二维码并提示重扫；手机重扫①即可。证书会自动为新 IP 重建（重启中继后完全生效）。
