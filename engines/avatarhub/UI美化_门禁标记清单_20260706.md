## 四·五、v3 动作按钮统一层（2026-07-08 按钮/面板美化）

> 背景：开播页面板内动作按钮长期散拼 `px-3 py-1.5 rounded bg-hub-blue/30 …`，大小不一；窄 grid 格里中文按钮被压成两行甚至竖排。本轮在 brand.css 新增 `.bd-act` 统一层并重构视觉效果面板布局。

- **`.bd-act` 基座**（brand.css v3 段）：高 32px / 字 13px / 圆角 10px / `white-space:nowrap` / `flex:none`（不被 grid/flex 拉伸压竖排）；`.sm`=26px 紧凑档；label 型上传按钮用 `.off` 表达禁用（label 无 `:disabled`）。
- **语义变体**：`primary`=品牌渐变+辉光（每子块唯一主动作：应用背景/保存离席画面/开始增强/保存映射/一键定妆包/写入角色底片/开始变声/保存效果配置）；`soft`=品牌蓝浅底（试音/试听/应用变声配置）；`ghost`=描边工具类（上传/切换/取色/全部入队）；`amber/pink/purple/green/red` 保留业务语义色（实验/妆容/变声/成功/危险）。
- **布局模式**：子块统一「`bd-sub-title`（图标+标题+bd-hint 说明）→ `bd-form`（bd-field 输入项）→ `bd-act-row`（动作行 flex-wrap）」；按钮不再当 grid 单元格塞窄列——这是竖排问题的根因。
- **主 CTA 呼吸辉光**：`.bd-glow-pulse`（2.8s brightness 脉动）挂在「一键开播」（可点时）与「一键无线开播」上；`prefers-reduced-motion` 全局降级已覆盖。

- **交互行悬停（v3.9 起）**：**整行可点**（点击展开/执行）才给行 hover（`hover:border-hub-blue/30` 或命令面板式底色高亮）；行内只有按钮可点的列表行不加行 hover（假承诺）。
- 卡片网格瓷砖内主操作（角色卡 启用/对话）保留**胶囊组**（与头像圆形语言呼应）；入口导航 tile（设置页快捷操作五宫格）保留 tile——两者语义分别是「卡内主动线」「页面入口」，不并入动作行体系。
