## 0. 范围与分工（先对齐认知）

**本项目只负责官网这四个版块**（= 无界科技 BOUNDLESS 的 幻声 VoiceX + 幻影 LiveX + 幻颜 FaceX）：

> ✅ **声音克隆** · ✅ **数字人 / 虚拟主播** · ✅ **实时直播换脸换声** · ✅ **图片 / 视频换脸**

这四块的官网文案与参数，**以本文件为准**核对修改（见 §1–§5）。

**以下版块不属于本项目，由其他项目/另一台 Cursor 负责，本方案不覆盖**：
- 智聊 ChatX（多平台 IM 自动成交、统一收件箱 TG/LINE/WhatsApp/Messenger）
- 通译 LingoX 的「聊天聚合 / 双向同传 IM」部分
- 无界底座（无审查 / 无禁区 私有大模型）

> 这三块请勿与上面四块混在同一产品页里用"本引擎能力"背书；它们各自有独立系统与负责人。本文件中凡标 ❌ 的条目，都属于"交给其他项目"，本项目不实现、也不承诺。

## 2. 必须修改 / 澄清的具体点（按优先级）

1. **【高】「看不出是换的脸和声音 / 看不出你是外国人 / 无痕」类措辞**
   - 现实：本引擎**默认强制 C2PA 水印 + AI 生成标识**（`PROVENANCE_ENABLED` 默认开），只有 `pro` 授权档 `watermark_free=true` 才去水印。
   - 建议官网改为：「真实自然、几乎以假乱真」即可；如要承诺「无水印」需明确「仅 pro 档/特定授权」，不要泛化承诺「完全无痕」。

2. **【高】智聊 ChatX / IM 聚合 / 自动成交**
   - 这些**不是本数字人引擎**的能力。若公司确有独立 IM 系统，官网应在产品页注明「由 ChatX 独立系统提供」，与「数字人/声音/换脸引擎」解耦；交付时按系统分别对接，避免售前承诺落空。

3. **【中】无审查叙事**
   - 本引擎自带安全闸门 + 合规水印。官网「无审查/无禁区」属另一条产品线（无界底座）。建议把「合规可溯源（C2PA/水印/可对外验真）」单独做成**信任卖点页**，与「无审查私有模型」物理分开，避免同一客户看到自相矛盾的两套叙事。

4. **【中】帧率 / 延迟硬指标**
   - 官网「32FPS · 38ms」是某机型示意。本引擎实测：口型 ~17–25ms/帧、首帧 ~0.9s、对话首音命中缓存 ~150ms。**建议官网标注「实测随硬件而定」并给区间**，勿用单一绝对值（不同卡差异巨大）。

5. **【低】品牌名统一**：见 §5，官网与引擎共用同一套品牌命名与配色。

## 3. 硬件配置对齐（官网推荐 vs 引擎实测）

官网现推荐：入门 RTX 4060Ti16G/4070、专业 4090、旗舰双卡。需按能力分档澄清：

| 能力 | 最低可用 | 建议 | 说明 |
|---|---|---|---|
| 图片/视频换脸（幻颜 FaceX） | 4060Ti 16G / 4070 | 4090 | 离线成片，低配可跑 |
| 实时换脸直播（幻影 LiveX·换脸） | 4070 / 4080 | 4090 | **720p 高清脸默认；1080p 需「超清档」+脸区原生通道**（4070 实测并发均值 323ms 达标）；口播极致档(CodeFormer) 约 5fps |
| 虚拟背景（叠加换脸） | 同上 | 同上 | CPU 抠像 ~5ms/帧@720p，不占显卡预算 |
| **实时高清数字人（活体+口型+克隆音，幻影 LiveX）** | 4090 24G | **RTX 5090 32G** | 本项目实时 25fps 高清是在 **5090(sm_120)** 上调优达成；4090 可跑标清/降档 |
| 同驻 本地大模型 + TTS + 口型 | — | 5090 32G 或多卡 | 14B LLM + TTS + 口型同驻需 ~30G 显存 |
| 同传 LingoX（克隆同传+字幕） | 5090 单机 或 分机 | 5090 + 分机 | STT/NMT 可分机（`cluster_map.json` 拓扑向导一键生成配置） |

→ 官网「入门 4060Ti 跑数字人」需谨慎：入门档更适合**换脸/图片**，**实时高清数字人请引导到 4090/5090**。
→ **实测口径速查（写官网参数时直接引用区间，勿编数）**：换脸引擎单帧 hd+GFPGAN ~230–260ms(4070·脸区裁剪)、CodeFormer ~320–360ms；3 路并发聚合 GFPGAN 8.8fps / CodeFormer 5.5fps；脸区清晰度(裁剪通道) 4.5×；口型 ~17–25ms/帧；对话首音命中缓存 ~150ms。
→ **2026-07-06 起本表有运行时版本**：引擎 `GET /api/hardware/guide` 按部署机 GPU 实测给「档位 + 每功能
推荐/降档/不建议」结论（控制台「设置 → 🧮 硬件档位」可视）；售前/官网写「推荐配置」时以该口径为准，
与《使用说明.md》§11 速查表、《销售一页纸_BOUNDLESS.md》三处同源。

## 4. 两套对外口径必须物理分开（合规风险提示）

本引擎内置：C2PA 标准嵌入、Ed25519 可对外验真、不可见水印、AI 生成检测、克隆伦理校验。这是**合规/可信**卖点。

官网当前混合了「无审查 / 看不出 / 换脸冒充」等叙事。**建议**：
- 面向**合规/政企/教育**客户：主打「本地不出网 + C2PA 可验真 + 安全闸门」（用本引擎能力背书）。
- 面向**出海/换脸**客户：归到 FaceX/LiveX 产品页，但**仍保留授权与免责声明**（官网底部已有，good）。
- 两类客户不要落到同一个产品页，避免「既合规又无审查」的自相矛盾。
