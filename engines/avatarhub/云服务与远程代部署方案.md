# 云服务与远程代部署方案

> 回答三个问题：① 远程代部署能否替代签名证书；② 不做本地部署、改用云服务是否可行；③ 换脸 / 实时翻译 / 变声三个功能上云的具体可行性与路线。

## 一、远程代部署服务（能省签名证书吗？）

**结论：能省一部分，但不能完全替代；建议两条腿走路。**

| 交付方式 | 是否需要签名证书 | 说明 |
| --- | --- | --- |
| 远程代部署（AnyDesk/ToDesk 上门装） | **不需要** | 我们亲手操作，SmartScreen 告警我们自己点掉，客户全程无感 |
| 私有部署（进客户机房/服务器） | 不需要 | 同上，且本来就是人工实施 |
| 官网自助下载安装 | **仍需要**（长期） | 用户自己下载时 SmartScreen「未知发布者」会吓退相当比例小白用户 |

**远程代部署产品化建议**（可直接进订单面板加 SKU）：

- **标准代部署**：99 USDT/次 — 远程装好客户端 + 环境 + 跑通第一个 demo（约 1 小时）
- **私有部署实施**：1,280 USDT 起（已在定价体系内）— 服务器级安装 + 调优 + 验收
- 附加价值：代部署顺便完成机器指纹采集 → 当场签发许可证，开通零等待
- 对现阶段（单量小、客单价高）来说，**代部署的转化率比签名证书更高**，证书可以推迟到月下载量上百再买

## 二、云服务化总体结论

**可行，且建议做成「本地 + 云」双模式，而不是二选一。**

核心账：一张 4090 级云卡约 0.3–0.6 USD/小时（RunPod/Vast.ai）或 ¥1.5–2.5/小时（AutoDL/仙宫云，国内延迟更优）。
按新定价 Pro 249 USDT/月含额度换算，只要把**按分钟计费定到 ≥ 3× GPU 成本**，毛利就是健康的。

三个功能按「上云难度」排序：**变声 < 实时翻译 < 实时换脸**。差别只在一件事——**延迟与带宽**：

| 功能 | 数据流 | 带宽 | 延迟容忍 | 云化可行性 |
| --- | --- | --- | --- | --- |
| 变声（RVC 类） | 纯音频 双向 | ~64 kbps | <300 ms 可接受 | ★★★★★ 立即可做 |
| 实时翻译（ASR→MT→TTS） | 纯音频 双向 | ~64 kbps | 链路本身 1–2 s，云端加 100–200 ms 无感 | ★★★★★ 立即可做 |
| 换脸（离线视频） | 上传→处理→下载 | 不敏感 | 无实时要求 | ★★★★★ 最容易，先做 |
| 换脸（实时直播） | 视频 双向 | 2–8 Mbps | **<200 ms 硬要求** | ★★★☆☆ 可做但挑机房 |

### 各功能要点

**1. 变声（最简单）**
- 音频包小，WebSocket/WebRTC 推流即可；RVC 类模型单卡可跑 4–8 路并发。
- 客户端只需麦克风采集 + 播放，Mac 版轻量控制台直接就能用——**这正好补齐了 Mac 无 N 卡的短板**。
- 计费：按分钟，tier 含免费分钟数（与已定的计量方案一致）。

**2. 实时翻译**
- 链路（ASR→翻译→克隆音色 TTS）本身就有 1–2 秒延迟，云端网络开销占比小，用户无感。
- 单卡可服务多路；翻译一段可调 DeepSeek API（官网已有 key）进一步降低自建成本。
- 会议/直播出海场景客户本来就没有 N 卡，**云版才是这个功能的主流形态**。

**3. 换脸**
- **第一步做离线云换脸**：网页/客户端上传视频 → 云端队列处理 → 出片下载。零延迟压力、排队即可撑高并发，单卡利用率最高，最先赚钱。
- **实时云换脸**：技术可行（WebRTC + 云 GPU），但要求用户到机房 RTT < 80 ms——国内用户配国内 GPU 云（AutoDL/阿里云），出海用户配新加坡/日本节点。头部主播专属实例（独占卡 + 固定节点）按小时/包月卖，普通用户仍推荐本地部署或代部署。

## 三、推荐架构（复用现有代码，不推倒重来）

```
用户端（Windows 薄客户端 / Mac 轻控制台 / 网页）
        │  WebSocket(音频) / WebRTC(视频) / HTTP(离线任务)
        ▼
网关（现有 FastAPI Hub 加一层：鉴权=许可证/订单 → 计量扣额度 → 任务路由）
        ▼
云 GPU 工作池（按需租：AutoDL/阿里云=国内，RunPod/Vast=海外）
  每台跑现有引擎（换脸/TTS/变声服务本来就是独立微服务，直接部署）
```

- 现有微服务架构天然适合搬上云：**每个服务本来就是独立进程 + HTTP 接口**，云端跑同一套代码。
- 网关加两件事即可：许可证/订单鉴权（已有 license.py + orders_web.jsonl）、按分钟/字符计量（P1 中间件，正好一份代码管本地额度与云额度）。
- GPU 池先**手动开 1–2 台包月卡**（AutoDL 4090 约 ¥1,300/月）验证需求，跑满再做自动扩缩容。
