## 已上生产（18799，2026-07-20 20:15 升级，47s 窗口，验收 22 项 ALL PASS + PERF OK）

### 功能面
- **CmdK 命令面板**（Ctrl/Cmd+K，全壳层）：页面/动作/本地会话/全局搜索（消息组优先、
  联系人独立组、`<em>` 高亮、空查询=最近会话）；带参命令 `@`转坐席 · `#`标签筛选 ·
  `>`页面直达 · `s 30m` 搁置
- **搜中即达**：搜索命中带 message_id，打开会话滚动+高亮气泡；不在首屏自动翻历史
  （generation token 防串会话），翻尽未命中明确 toast；深链 `?conv=&mid=`（兼容 `?focus=`）
- **会话列表**：游标分页（store `before_ts`）+ 滚动到底自动加载；100 条首屏；
  500 条压测首渲 ~45ms 无瓶颈
- **转派闭环**：批量转派修静默空转（P23 起 `updated` 恒 0 的真 bug）+ viewer 拒写 +
  审计留痕 + 被转派坐席实时提醒（SSE 铃铛+提示音+可点跳）+ 处理人徽章事件驱动热更新
- **通知中心**：打开会话即消对应通知；类型筛选 chips；转派通知可点跳
- **移动端**：底部平台条（FIXED_PLATS 早绘）、软键盘适配（interactive-widget +
  visualViewport 兜底）、窄屏顶栏图标化、≤560 仅留 SLA 药丸
- **三语**：vi = en 底 + VI 覆盖（缺键回英文），坐席端+概览页 ~600 键；语言循环 zh→en→vi
- **KB**：电商起步包 5→11 条（含英文触发词），FAQ 解决率门禁 25%→100%
