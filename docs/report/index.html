05 / PROGRESS

近期重点工作（2026-07）

确立"打破五界"品牌叙事与"让沟通，无界"Slogan；旧品牌（华灵/华影/灵犀）作为 SEO 承接。

三系产品命名体系落地 已落地

体系化为智连 / 幻境 / 通达三系 + "…X"后缀；新增智拓 ReachX、通传 VoxX，补齐 7 产品；全站 11 处文件文案统一（UI + Bot 话术 + system prompt），避免口径分裂。

官网信息架构方案 方案就绪

从"单页锚点"升级为"多产品站"：类别导航 + 独立落地页 + 数据驱动陈列；规划新增 /translate、/avatar、/dubbing 落地页。

品牌资产库建设 已落地

统一 logo / favicon / 单色标、7 款产品专属图标、公司与产品锁定组合（lockups）、多平台社媒背景图与品牌字体。

私域化与抗封接入 进行中

域名防封策略、Cloudflare 抗封接入清单，保障获客与承接链路稳定。

工程瘦身与清理 已落地

两大主项目从 17+ GB 隔离瘦身至可维护体积（智聊 15.68GB→470MB，获客 1.58GB→349MB），运行产物入隔离区、可恢复。
